# ProjectMK Agent Notes

Before making code changes in this repository, read these files first:

- `AGENT/CODING_CONVENTION.md`
- `AGENT/CODING_CONVENTION.txt`

Rules:

- `AGENT/CODING_CONVENTION.md` is the canonical project convention document.
- `AGENT/CODING_CONVENTION.txt` must be kept in sync as a human-friendly companion summary.
- When either convention changes, update both files in the same task.
